ℹ About SmartShiftAI
SmartShiftAI is an AI-powered workforce planning system built as a Final Year Project.
It uses Gradient Boosting and Ridge Regression to forecast customer demand, then generates optimised shift schedules to minimise labour cost while maintaining service levels.

Pipeline: Upload CSV → Train ML Model → Demand Forecast → Shift Scheduling → Cost Analysis → Report Generation

Built with Flask · scikit-learn · Chart.js · Python 3.14
